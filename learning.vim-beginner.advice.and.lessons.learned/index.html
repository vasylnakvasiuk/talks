Operator + Text object

Operators


  • d – delete (also cut)
  • c – change (delete, then place in insert mode)
  • y – yank (copy)
  • v – visually select

  • gu – make lowercase
  • gU – make uppercase
  • g~ – swap case

Text objects


Text object = prefix + object

Text objects


Text object prefix

  • a – select "a"n object
  • i – select an "inner" object

Text objects


Object

  • w – word
  • W – WORD
  • s – sentence
  • p – paragraph
  • ]   [ – [] block
  • )   (   b – () block
  • >   < – <> block
  • }   {   B – {} block
  • t – tag block
  • '   "   ` – quoted string
